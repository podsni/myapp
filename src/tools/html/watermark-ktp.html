Tentang & Kebijakan Privasi

Watermark KTP adalah alat berbasis web gratis dan aman untuk menambahkan watermark pada scan KTP Anda. Watermark berupa teks berisi tanggal dan tujuan scan KTP — misalnya "Verifikasi e-wallet, 10-10-2021" — untuk mencegah penyalahgunaan dan penipuan data.

Privasi penuh. Semua proses berjalan 100% di sisi Anda (client-side). Tidak ada gambar yang pernah dikirim keluar dari browser Anda — tidak ada server, upload, atau penyimpanan.

Cara pakai

  • Tarik / pilih / tempel gambar KTP.
  • Seret watermark untuk memindahkan.
  • Tarik bulatan di atas untuk memutar.
  • Tarik sudut kotak untuk mengubah ukuran.
  • Klik dua kali watermark untuk mengedit teks.
  • Unduh sebagai PNG resolusi penuh.

Kebijakan Privasi

  • Tidak pernah menjual atau menyalahgunakan data KTP siapa pun.
  • Tidak ada file yang dikirim keluar dari browser Anda.
  • Semua pemrosesan dilakukan sepenuhnya di perangkat Anda.
  • Berkomitmen pada keamanan data pribadi di dunia digital.
Dibuat untuk keamanan data pribadi • Tidak ada data yang meninggalkan perangkat Anda
